# Conduit Next.js 提出課題

[RealWorld](https://realworld-docs.netlify.app/docs/specs/backend-specs/introduction) の仕様を満たすウェブアプリ

デプロイ先: https://conduit-next.vercel.app/

## 作成したページ
- Home
- Create/Edit Article
- Article
- Authentication
- Profile
- Settings

## 実装した機能
- ユーザーの登録・編集・ログイン
- 記事の投稿・編集・削除
- 著者
- タグ付け機能
- 記事へのコメント投稿・削除
